CodeDoor Style Examples

We now have a nice standardized style system where styles are broken down into classes corresponding to different components. To understand how to use these classes, let's look at some examples.

Buttons

<div class="btn btn-sm">Small Button</div>
<div class="btn btn-md">Medium Button</div>
